# DevOps Automation Toolkit

A collection of automation scripts built as part of a hands-on DevOps learning project.

## Scripts
- auto_update.sh — System updates
- daily_backup.sh — Automated backups
- docker_prune.sh — Docker cleanup
- repo_sync.sh — GitHub sync
- log_rotation.sh — Log rotation
- vuln_scan.sh — Vulnerability scanning
- db_dump.sh — Database dumps
- s3_sync.sh — AWS S3 sync
- ec2_health.sh — EC2 health checks
- til.sh — Today I Learned logger
- drift_detect.sh — Terraform drift detection

## Tools Used
- Bash, Python, AWS CLI, Terraform, Docker
